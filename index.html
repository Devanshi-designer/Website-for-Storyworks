    Problem

    Teams waste hours normalizing file names and structure: scattered image names, ambiguous final.mov files, missing context — blocking editors and slowing delivery.

    Solution — LABEL

    Ingest a client folder. LABEL analyzes content, extracts semantic tags, maps to client naming templates, and produces a verified set of renamed files plus a CSV mapping report for audit and handoff.

    LABEL keeps creative control human while automating repetitive work.

    It runs scene detection, speaker-aware transcription, OCR on frames, image classification and metadata extraction. Outputs are mapped into customizable naming templates (JSON/YAML). An optional verification UI lets editors approve or tweak names before final export. LABEL supports batching, retry logic and produces human-readable CSV and JSON logs for downstream systems.
